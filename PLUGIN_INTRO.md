# Joplin 插件仓库项目简介

以下提供中英文两个版本，可直接复制用于 Joplin 插件发布页面、npm 描述或社区分享帖。

---

## 中文版

**Full Notebook View** 是一款 Joplin 侧边栏插件，为你带来类似 Windows 资源管理器的笔记本树形视图体验。

插件提供了双标签页侧边栏：**Notebooks** 页以层级树展示所有笔记本与笔记，支持展开/折叠、搜索、排序、日期筛选与排除管理；**Outline** 页自动提取当前笔记的 Markdown 标题大纲，点击即可快速跳转。右键菜单涵盖新建、重命名、移动、导出（Markdown / HTML / PDF / JEX）、复制链接等完整操作。底部还集成了一键同步按钮与同步日志面板，并支持隐藏笔记标题栏以获得更沉浸的编辑体验。

无论你是需要管理大量笔记本的重度用户，还是希望在侧边栏快速定位笔记和大纲的写作者，这款插件都能显著提升你的 Joplin 使用效率。

**兼容性**：Joplin v3.5+

## English Version

**Full Notebook View** is a Joplin sidebar plugin that brings a Windows Explorer–style notebook tree view to your workspace.

It provides a dual-tab sidebar: the **Notebooks** tab displays all notebooks and notes in a hierarchical tree with expand/collapse, search, sorting, date filtering, and exclusion management; the **Outline** tab auto-extracts Markdown headings from the current note for quick navigation. The context menu covers create, rename, move, export (Markdown / HTML / PDF / JEX), copy link, and more. A one-click sync button with a sync log panel is integrated at the bottom, and you can optionally hide the note title bar for a more immersive editing experience.

Whether you manage a large number of notebooks or simply want faster navigation and outlining, this plugin is designed to boost your Joplin productivity.

**Compatibility**: Joplin v3.5+
